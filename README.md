# yagb-emu (IN DEVELOPMENT) 🚀
Yet another GameBoy Emulator!

## Build 🔨
Use cmake to configure and build the project.

```bash
mkdir build                 # Create a Build Directory
cd build && cmake ..        # Run cmake to configure and build
```
